- PM-09：修語音 `not-allowed` — 注入頁面授權按鈕解 user gesture 問題

  - 浮層「允許麥克風」按鈕的 click 才是有效 user gesture → `getUserMedia` 彈標準授權 → 啟動語音
  - 「跳過」按鈕可不錄語音、不阻擋 DOM/console/network；只動 `inject.ts`
