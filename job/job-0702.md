**背景**：日票 NT$20，付款成功起算 24 小時，到期自動降回免費版。用綠界一次性付款（非定期定額），信用卡 + 非信用卡都能付。

`users` 表新增欄位（FOX 需在 SQL Editor 手動跑）：

```sql

```

`NULL` = 沒有日票；有值 = 日票到期時間。

#### §2 Server：新增日票付款端點

```
POST /api/day-pass/create
```

- 需要登入（從 auth header 取 user_id）
- 建立綠界一次性付款訂單
- 訂單金額 NT$20
- TradeDesc：「BugEzy 日票（24小時）」
- 付款方式：ALL（信用卡 + ATM + 超商）

```ts
async function handleDayPassCreate(request: Request, env: Env): Promise<Response> {
  // 1. 驗證登入
  const user = await getAuthUser(request, env);
  if (!user) return json({ error: '請先登入' }, 401);

  // 2. 檢查是否已有日票或月費
  const { data } = await supabase(env)
    .from('users')
    .select('plan, day_pass_expires_at')
    .eq('user_id', user.user_id)
    .single();

  if (data?.plan === 'paid') {
    return json({ error: '您已是月費用戶，不需購買日票' }, 400);
  }
  if (data?.day_pass_expires_at && new Date(data.day_pass_expires_at) > new Date()) {
    return json({ error: '您已有有效日票，到期後才能再購買' }, 400);
  }

  // 3. 建立綠界訂單（一次性付款，非定期定額）
  const tradeNo = 'DP' + Date.now(); // 唯一訂單編號
  const params = {
    MerchantID: env.ECPAY_MERCHANT_ID,
    MerchantTradeNo: tradeNo,
    MerchantTradeDate: formatECPayDate(new Date()),
    PaymentType: 'aio',
    TotalAmount: '20',
    TradeDesc: 'BugEzy 日票（24小時無限使用）',
    ItemName: 'BugEzy 日票 NT$20',
    ReturnURL: `${getBaseUrl(request)}/api/day-pass/callback`,
    ClientBackURL: `${getBaseUrl(request)}/day-pass-success`,
    ChoosePayment: 'ALL',
    EncryptType: '1',
    CustomField1: user.user_id, // 用來在 callback 識別使用者
  };

  // 4. 產生檢查碼 + 回傳綠界付款 URL
  // （跟現有月費的 ECPay 串接邏輯一樣，複用 generateCheckMacValue）
  const checkMac = generateCheckMacValue(params, env.ECPAY_HASH_KEY, env.ECPAY_HASH_IV);
  params.CheckMacValue = checkMac;

  // 5. 回傳 form HTML（auto-submit 到綠界）
  return new Response(buildECPayFormHTML(params, env.ECPAY_PAYMENT_URL), {
    headers: { 'Content-Type': 'text/html; charset=utf-8' },
  });
}
```

#### §3 Server：日票 callback

```
POST /api/day-pass/callback
```

綠界付款成功後打回來：

```ts
async function handleDayPassCallback(request: Request, env: Env): Promise<Response> {
  const body = await request.formData();
  const params = Object.fromEntries(body.entries());

  // 1. 驗證 CheckMacValue
  if (!verifyCheckMacValue(params, env.ECPAY_HASH_KEY, env.ECPAY_HASH_IV)) {
    return new Response('CheckMacValue Error', { status: 400 });
  }

  // 2. 確認付款成功（RtnCode === '1'）
  if (params.RtnCode !== '1') {
    return new Response('1|OK'); // 綠界要求回 1|OK
  }

  // 3. 取 user_id（從 CustomField1）
  const userId = params.CustomField1 as string;

  // 4. 設定日票到期時間（24 小時後）
  const expiresAt = new Date(Date.now() + 24 * 60 * 60 * 1000).toISOString();

  await supabase(env)
    .from('users')
    .update({
      day_pass_expires_at: expiresAt,
      plan: 'day_pass', // 新增 plan 值
    })
    .eq('user_id', userId);

  return new Response('1|OK');
}
```

#### §5 Server：plan 判斷邏輯更新

現有判斷 `plan === 'paid'` 的地方，都要加上日票判斷。新增 helper：

```ts
function isActiveUser(user: { plan: string; day_pass_expires_at?: string | null }): boolean {
  if (user.plan === 'paid') return true;
  if (user.plan === 'day_pass' && user.day_pass_expires_at) {
    return new Date(user.day_pass_expires_at) > new Date();
  }
  return false;
}
```

搜尋 server 所有 `plan === 'paid'` 或 `plan !== 'free'` 的判斷，全部改用 `isActiveUser()`。

#### §6 Server：日票到期自動降回

在現有的 cron 或 API 請求時檢查：如果 `day_pass_expires_at` 已過期，把 plan 改回 `free`：

```ts
// 可以在每次 API 請求的 auth middleware 順便檢查
if (user.plan === 'day_pass' && user.day_pass_expires_at) {
  if (new Date(user.day_pass_expires_at) <= new Date()) {
    // 到期：降回免費
    await supabase(env).from('users').update({
      plan: 'free',
      day_pass_expires_at: null,
    }).eq('user_id', user.user_id);
    user.plan = 'free';
  }
}
```

免費版次數不重置（不動 usage 欄位）。
