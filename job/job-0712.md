### 🟡 PM-228：Official MCP Registry 提交

**背景**：Official MCP Registry（https://github.com/modelcontextprotocol/registry）是 PulseMCP 等目錄的上游資料源。提交後會自動被多個目錄收錄。

**做法**：使用 `mcp-publisher` CLI 工具發布 BugEzy 的 `server.json` 到 Official MCP Registry。

**步驟**：

1. 先研究 Official MCP Registry 的提交流程：
   - 讀 https://github.com/modelcontextprotocol/registry 的 README 和 docs/
   - 確認 server.json 格式要求
   - 確認是否需要 Dockerfile 或只需 remote endpoint

2. 在 `C:\dev\bugezy\server\` 建立符合規範的 `server.json`：
   - name: bugezy
   - description: Voice-powered bug reporting with 13 MCP tools
   - endpoint: https://bugezy.dev/mcp
   - transport: streamable-http
   - tools: 13 個工具名稱
   - repository: https://github.com/fox100039-design/bugezy

3. 用 CLI 工具提交（如果需要 npm install 某個工具，先安裝）

4. 如果 CLI 方式太複雜，改用 GitHub PR 方式直接提交 server.json 到 registry repo

**注意**：BugEzy 是 hosted remote server（不是 stdio 本地跑的），確認提交方式支援 remote endpoint。

**驗收條件**：
1. 成功提交到 Official MCP Registry（PR 開出或 CLI 發布成功）
2. 把提交連結或 PR 連結回報

### 做法（研究 → 決策 → 執行）
1. **研究**：官方 registry 用 `mcp-publisher` CLI（**無 GitHub PR 途徑**）；remote server 用 `remotes:[{type:streamable-http,url}]`；namespace 需域名驗證（GitHub OIDC / DNS / HTTP）。
2. **踩雷 → 換路**：先試 GitHub Actions OIDC（namespace `io.github.fox100039-design/bugezy`）→ **推 workflow 被 GitHub 擋**（本機 PAT 無 `workflow` scope，無法建立 `.github/workflows/*`）。改用**HTTP 域名驗證**（namespace `dev.bugezy/bugezy`，品牌更好且我可自主完成——BugEzy 就是 hosted remote server，我掌控 Worker）。
3. **執行**：
   - `server/server.json`（`$schema` + name `dev.bugezy/bugezy` + title/description〔≤100 字〕/version 1.1.3 + repository + websiteUrl + remotes streamable-http `https://bugezy.dev/mcp`）。
   - Ed25519 金鑰對（openssl）→ Worker 新增 `GET /.well-known/mcp-registry-auth` serve 公鑰（`v=MCPv1; k=ed25519; p=…`，`no-store`）→ deploy `cb2a8400` → 線上驗證一致回傳。
   - `mcp-publisher login http --domain=bugezy.dev --private-key=<hex>` → ✓ Successfully logged in；`mcp-publisher publish` → 首次 422（description >100）→ 縮短至 96 字 → ✓ Successfully published。
   - **私鑰 / mcp-publisher.exe / token 皆本機用完即刪，未進 repo**（公鑰留在 Worker route，安全）。

### 驗收
1. 成功提交（CLI 發布成功 + 註冊 API 實查 active）✅
2. 提交結果：**Official MCP Registry `dev.bugezy/bugezy` v1.1.3**（查詢：`https://registry.modelcontextprotocol.io/v0/servers?search=bugezy`）✅
