#### Server 路由

```ts
if (url.pathname === '/' && request.method === 'GET') {
  return new Response(HOMEPAGE_HTML, {
    headers: { 'Content-Type': 'text/html; charset=utf-8' },
  });
}
```

加在所有其他路由之前。
