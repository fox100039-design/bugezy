### 🟡 PM-250：annotate.ts Whisper 提示文字 i18n（四處硬寫中文）

**背景**：FOX 實測截圖精準轉錄（Whisper），發現錄音中的提示字（「Whisper 錄音中，講完按 ⏹ 轉錄…」）不管切到哪個語言都是中文。PM-248 修 7 只修了 Web Speech 區塊的字幕條 i18n，Whisper 區塊遺漏。

**四處硬寫中文需改 i18n**：

1. **第 651 行**：`'🔴 Whisper 錄音中，講完按 ⏹ 轉錄…'`
   → i18n key（如 `an-whisper-recording`）

2. **第 686 行**：`'⏳ Whisper 轉錄中…'`
   → i18n key（如 `an-whisper-transcribing`）

3. **第 703 行**：`'Whisper 為付費功能，請升級（或改用鍵盤）'`
   → i18n key（如 `an-whisper-paid-only`）

4. **第 752 行**：`'🎙️ 付費版 Whisper：按 🎤 錄音描述，講完按 ⏹ 轉錄'`
   → i18n key（如 `an-whisper-prompt`）

**修法**：

1. **i18n.ts 新增四個 key**，七語各自翻譯：
   - zh：保持目前中文
   - zh-CN：簡體版（可用 toSimplified 或直接手寫）
   - en：英文
   - ja：日文
   - ko：韓文
   - vi：越南文
   - yue：跟 zh 相同即可

2. **annotate.ts**：四處硬寫中文改為 `t(key, annotateUILang)` 呼叫。

**注意**：annotate.ts 是擴充頁，有 `annotateUILang` 變數（第 27 行），已在 PM-248 修 7 用過，直接復用。

**改動檔案**：`extension/src/annotate.ts` + `extension/src/i18n.ts`

**驗收條件**：
1. 日文模式截圖 Whisper → 提示文字為日文
2. 英文模式 → 英文
3. 簡體中文 → 簡體
4. 各語提示文字正確對應
5. `npm run build` 無 TS 錯誤

完成後在下方寫 `🔵 DONE-250`。
