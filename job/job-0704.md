### 🟡 PM-136：SEO — sitemap.xml + robots.txt + meta tags（搜尋引擎收錄）

**背景**：bugezy.dev 上線一週，各大搜尋引擎完全搜不到。需要主動提交 sitemap 讓 Google/Bing 收錄。

**目標**：server 新增 `/sitemap.xml` + `/robots.txt` 端點 + 所有頁面加 SEO meta tags。

---

#### §1 Server：新增 GET /sitemap.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url><loc>https://bugezy.dev/</loc><changefreq>weekly</changefreq><priority>1.0</priority></url>
  <url><loc>https://bugezy.dev/install</loc><changefreq>monthly</changefreq><priority>0.9</priority></url>
  <url><loc>https://bugezy.dev/features</loc><changefreq>monthly</changefreq><priority>0.8</priority></url>
  <url><loc>https://bugezy.dev/changelog</loc><changefreq>weekly</changefreq><priority>0.7</priority></url>
  <url><loc>https://bugezy.dev/guide</loc><changefreq>monthly</changefreq><priority>0.6</priority></url>
  <url><loc>https://bugezy.dev/faq</loc><changefreq>monthly</changefreq><priority>0.5</priority></url>
  <url><loc>https://bugezy.dev/privacy</loc><changefreq>yearly</changefreq><priority>0.3</priority></url>
</urlset>
```

回傳 Content-Type: `application/xml`。

### 🟡 PM-137：語言選擇 — popup 語系切換（粵語/日/韓/英/越）

**背景**：Groq Whisper 同一模型支援 99 種語言，只需改 `language` 參數。Web Speech API 同理改 `lang`。popup 加語言選擇讓使用者切換，即可打開亞洲全市場。

---

| 🇹🇼 台灣 | 繁體中文 | `zh` | `zh-TW` |
| 🇭🇰 香港 | 粵語 | `yue` | `zh-HK` |

#### §3 popup.ts：存語言設定

```ts
const LANG_KEY = 'bugezy:language';

const langSelect = document.getElementById('lang-select') as HTMLSelectElement;

// 初始化
const langStore = await chrome.storage.local.get(LANG_KEY);
langSelect.value = langStore[LANG_KEY] || 'zh';

// 切換時存
langSelect.addEventListener('change', async () => {
  await chrome.storage.local.set({ [LANG_KEY]: langSelect.value });
});
```

---

#### §4 background.ts：Whisper 語音轉錄讀取語言設定

找到呼叫 `/api/transcribe` 的地方，把 form 的 language 參數改從 storage 讀取：

```ts
const langStore = await chrome.storage.local.get('bugezy:language');
const whisperLang = langStore['bugezy:language'] || 'zh';
form.append('language', whisperLang);
```

---

#### §5 Server：handleTranscribe 讀取 language 參數

目前 server 的 transcribe 可能寫死 `language: 'zh'`，改為從 form 讀取：

```ts
const language = form.get('language')?.toString() || 'zh';

// 驗證（只接受白名單語言，防濫用）
const ALLOWED_LANGS = ['zh', 'yue', 'ja', 'ko', 'en', 'vi'];
const finalLang = ALLOWED_LANGS.includes(language) ? language : 'zh';

groqForm.append('language', finalLang);
```

---

#### §6 inject.ts / content.ts：Web Speech API 讀取語言設定

找到 Web Speech API 初始化的地方（`new SpeechRecognition()`），把 `recognition.lang` 改從 storage 讀取：

```ts
// 語言對照表
const LANG_MAP: Record<string, string> = {
  'zh': 'zh-TW',
  'yue': 'zh-HK',
  'ja': 'ja',
  'ko': 'ko',
  'en': 'en',
  'vi': 'vi',
};

const langStore = await chrome.storage.local.get('bugezy:language');
const lang = langStore['bugezy:language'] || 'zh';
recognition.lang = LANG_MAP[lang] || 'zh-TW';
```

注意：inject.ts 在 MAIN world 沒有 `chrome.storage`，需透過 content.ts → window.postMessage 或 data attribute 傳語言設定進來。
