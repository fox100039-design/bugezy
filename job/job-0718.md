3. **final handler 開頭加去重判斷**：
```typescript
if (res.isFinal) {
  clearErInterimPromote();
  let text = res[0].transcript;
  if (reportLang === 'zh-CN') text = toSimplified(text);

  // PM-246：去重——如果這段 final 文字在 5 秒內已被 stale interim 自動升級寫入過，跳過
```

4. 停錄（edit `stopVoice` / inject 主 STOP）與 inject `forceRestartVoice` 用完整 `clearXxxInterimPromote()`。

**修法**（`edit-report.ts`）：

搬入 PM-241 的 stale interim 自動升級邏輯，適配 edit-report 的文字框場景：

1. **在語音辨識建立函數裡新增狀態變數**：
```typescript
let erInterimTimer: ReturnType<typeof setTimeout> | null = null;
let erLastInterim = '';
```

2. **`onresult` 的 interim 區塊末尾加自動升級**：
```typescript
} else {
  let seg = res[0].transcript;
  if (reportLang === 'zh-CN') seg = toSimplified(seg);
  interim = seg;

  // PM-245：stale interim 3 秒未變 → 自動寫入文字框（粵語/越南語修復）
  if (seg !== erLastInterim) {
    erLastInterim = seg;
    if (erInterimTimer) clearTimeout(erInterimTimer);
    erInterimTimer = setTimeout(() => {
      if (erLastInterim && listening) {
        const cursorPos = descInput.selectionStart;
        const isAtEnd = cursorPos === descInput.value.length;
        descInput.value += erLastInterim;
        if (!isAtEnd) {
          descInput.selectionStart = cursorPos;
          descInput.selectionEnd = cursorPos;
        }
        voiceStatus.textContent = T('er-listening');
        erLastInterim = '';
      }
    }, 3000);
  }
}
```

3. **真 final 到達時清除 timer**（防其他語言重複，同 PM-241）：
```typescript
if (res.isFinal) {
  // 清除 stale interim timer（防重複）
  if (erInterimTimer) { clearTimeout(erInterimTimer); erInterimTimer = null; }
  erLastInterim = '';
  // ... 既有 final 邏輯
}
```

4. **停止語音時清除**：在 listening 設為 false 的地方加：
```typescript
if (erInterimTimer) { clearTimeout(erInterimTimer); erInterimTimer = null; }
erLastInterim = '';
```
