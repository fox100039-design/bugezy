### 🟡 PM-214：Whisper 轉錄強制輸出繁體中文（zh + yue）

**背景**：粵語語音測試結果——辨識準確度高 ✅，但輸出簡體字 ❌。中文模式偶爾也會輸出簡體。台灣市場產品必須統一繁體。

**問題根因**：Groq Whisper 的 `language` 參數只控制辨識語言，不控制輸出字體（簡/繁）。

**修法**：在 server 的 `/api/transcribe` 呼叫 Groq Whisper 時，對 `zh` 和 `yue` 語言加上 `prompt` 參數，引導輸出繁體中文。

**具體改動**（`server/src/index.ts` 的 transcribe 路由）：

找到呼叫 Groq Whisper API 的地方，在 FormData 加入 `prompt` 欄位：

```typescript
// 對中文和粵語，加 prompt 引導繁體輸出
if (language === 'zh' || language === 'yue') {
  formData.append('prompt', '以下是繁體中文的語音轉錄內容。');
}
```

**注意事項**：
- `prompt` 只對 `zh` 和 `yue` 加，英文 `en` 不需要
- `prompt` 不影響辨識準確度，只引導輸出字體風格
- 日韓越 (ja/ko/vi) 未來解鎖時不需要這個處理

**驗收條件**：
1. Deploy 後 FOX 用粵語影片重新測試，確認輸出繁體中文
2. 再用中文測一次，確認也是繁體
