#### §2 inject.ts 改動 — 背景循環緩存

**核心概念**：inject.ts 載入時就開始 rrweb 錄製 + console/network 攔截，但資料存在**循環 buffer**裡（只保留最近 30 秒）。

**新增背景模式**：

在 `main()` 的最上面（不需要等 START），自動開始緩存：

```ts
// ========== 背景循環緩存（⏪ 回溯用）==========
const REWIND_WINDOW = 30_000; // 30 秒
let bgEvents: { data: unknown; timestamp: number }[] = [];
let bgConsoleLogs: { data: ConsoleLog; timestamp: number }[] = [];
let bgNetworkErrors: { data: NetworkError; timestamp: number }[] = [];

// 啟動背景 rrweb 錄製
let bgStopRrweb: (() => void) | null = null;
try {
  const stop = record({
    emit(event: any) {
      const ts = event.timestamp || Date.now();
      bgEvents.push({ data: event, timestamp: ts });
    },
  });
  bgStopRrweb = stop ?? null;
  blog('背景緩存 rrweb 已啟動');
} catch (err) {
  blog('⚠ 背景 rrweb 啟動失敗', err);
}

// 每 5 秒清除超過 30 秒的舊事件
setInterval(() => {
  const cutoff = Date.now() - REWIND_WINDOW;
  bgEvents = bgEvents.filter(e => e.timestamp > cutoff);
  bgConsoleLogs = bgConsoleLogs.filter(e => e.timestamp > cutoff);
  bgNetworkErrors = bgNetworkErrors.filter(e => e.timestamp > cutoff);
}, 5000);
```

**console/network 攔截改動**：

目前的 console.warn/error 和 fetch/XHR 攔截只在 `recording === true` 時收集。改為**永遠收集到背景 buffer**，`recording === true` 時**也**存到錄製 buffer：

```ts
console.error = (...args) => {
  const entry = { level: 'error' as const, message: stringifyArgs(args), timestamp: Date.now() };
  
  // 永遠存背景 buffer（回溯用）
  bgConsoleLogs.push({ data: entry, timestamp: entry.timestamp });
  
  // 錄製中也存錄製 buffer
  if (recording) {
    consoleLogs.push(entry);
    post({ source: BUGEZY_SOURCE, dir: 'to-content', kind: 'FLUSH_CONSOLE', log: entry });
  }
  
  return originalError(...args);
};
```

Network（fetch/XHR）同理：永遠存 bgNetworkErrors，recording 時也存 networkErrors。

#### §4 startRecording() 與背景緩存的關係

使用者按「🎬 錄製」時，需要**停止背景 rrweb 再重新開始錄製用的 rrweb**（同一個頁面不能同時有兩個 rrweb record instance）：

function stopRecording() {

function restartBackgroundBuffer() {

  bgNetworkErrors = [];

let monitoringEnabled = false;
let monitorInterval: ReturnType<typeof setInterval> | null = null;

function startMonitoring() {
  if (monitorInterval) return;
  monitoringEnabled = true;
  monitorInterval = setInterval(async () => {
    try {
      const tab = await getActiveTab();
      if (!tab?.id) return;
      // 向 content → inject 要即時 errors
      const result = await chrome.tabs.sendMessage(tab.id, { type: 'GET_LIVE_ERRORS' });

            title: tab.title,
            consoleLogs: result.consoleLogs,
            networkErrors: result.networkErrors,
            timestamp: Date.now(),
          }),
        });
      }
    } catch {}
  }, 10000);
}

function stopMonitoring() {
  monitoringEnabled = false;
  if (monitorInterval) {
    clearInterval(monitorInterval);
    monitorInterval = null;
